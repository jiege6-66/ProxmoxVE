# 配置和部署指南

本目录包含使用 community-scripts 配置和部署 Proxmox VE 容器的综合指南。

## 📚 可用指南

### [配置参考](CONFIGURATION_REFERENCE.md)

构建系统中所有配置选项、环境变量和高级设置的完整参考。

**涵盖的主题：**

- 容器规格（CPU、RAM、磁盘）
- 网络配置（IPv4/IPv6、VLAN、MTU）
- 存储选择和管理
- 权限模式和功能
- 操作系统选择和版本

### [默认系统指南](DEFAULTS_SYSTEM_GUIDE.md)

理解和自定义容器部署的默认设置。

**涵盖的主题：**

- 默认系统设置
- 每个脚本的覆盖
- 自定义默认配置
- 环境变量优先级

### [无人值守部署](UNATTENDED_DEPLOYMENTS.md)

在无用户交互的情况下自动化容器部署。

**涵盖的主题：**

- 环境变量配置
- 批量部署
- CI/CD 集成
- 脚本化安装
- 预配置模板
