# cloud-init.func 函数参考

Cloud-init 和 VM 配置函数。

## 核心函数

### generate_cloud_init()
生成 cloud-init 配置。

### generate_user_data()
为 VM 生成用户数据脚本。

### apply_cloud_init()
将 cloud-init 应用于 VM。

### setup_ssh_keys()
部署 SSH 公钥。

### setup_static_ip()
在 VM 上配置静态 IP。

### setup_dns()
为 VM 配置 DNS。

### setup_ipv6()
在 VM 上启用 IPv6。
