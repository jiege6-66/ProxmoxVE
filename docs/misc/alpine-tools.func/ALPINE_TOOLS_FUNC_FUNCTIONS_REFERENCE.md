# alpine-tools.func 函数参考

Alpine 特定的工具安装函数。

## 核心函数

### apk_update()
更新 Alpine 包列表。

### apk_add(PACKAGES)
安装 Alpine 包。

### apk_del(PACKAGES)
删除 Alpine 包。

### add_community_repo()
启用 Alpine community 仓库。

### add_testing_repo()
启用 Alpine testing 仓库。

### Alpine 工具函数
- `setup_nodejs()` - Alpine Node.js
- `setup_php()` - Alpine PHP
- `setup_mariadb()` - Alpine MariaDB
- `setup_postgresql()` - Alpine PostgreSQL
- （+ 更多 Alpine 特定设置）
